| **区别**  | **成员变量**   | **局部变量**                |
|---------|------------|-------------------------|
| 类中位置不同  | 类中方法外      | 方法内或者方法申明上              |
| 内存中位置不同 | 堆内存        | 栈内存                     |
| 生命周期不同  | 随着对象的存在而存在 | 随着方法的调用而存在，随着方法的调用完毕而消失 |
| 初始化值不同  | 有默认的初始化值   | 没有默认的初始化值，必须先定义，赋值，才能使用 |
